預覽和發布 Web 體驗

在本主題中，您將學習如何預覽和發布Web Experience。

在開發過程中可以輕鬆預覽Web體驗。也可以共享預覽體驗的鏈接。體驗完成後，可以將其發布以使其公開可用。

預覽網絡體驗

預覽Web體驗可讓您查看體驗如何在移動，平板電腦和台式設備上顯示。要預覽Web Experience，請編輯該體驗，然後單擊預習在“體驗編輯器"中。

有兩個選項：

  • 響應式預覽 -在新的瀏覽器窗口中打開體驗以進行預覽
  • 獲取共享鏈接 -提供預覽網址，以便可以與他人共享預覽。

產生的可共用 URL 的有效期為 72 小時。

注意：如果您已將任何工作階段逾時設定為任何存取控制設定的一部分，72 小時逾時將覆寫您設定的工作階段逾時。發布體驗後，將強制執行會話超時（如果有）。

發佈網絡體驗

發布Web Experience使其可供查看者使用，並為該站點生成靜態URL。若要發佈經驗，請依照下列步驟執行：

注意：如果您最近變更視訊中繼資料或播放清單，這些變更可能需要幾分鐘才會出現在已發佈的體驗上。
  1. 單擊體驗名稱，以在“體驗編輯器"中打開該站點。
  2. 點擊發布。
  3. 在“發布確認"對話框上，單擊立即發布。
  4. 體驗將被公佈，並顯示網站 URL。

將更改發佈到Web體驗

當您更改體驗時，必須重新發布。需要重新發布的體驗將顯示“體驗狀態"為未發布的更改。

要重新發布更改，請按照發佈網絡體驗本主題的小節。

注意：如果未出現更改，您可能需要清除瀏覽器緩存和 cookie。

使網絡體驗脫機

離線使用體驗會使其無法在其 URL 中使用。要使體驗脫機，請單擊一行，然後單擊離線。要確認該體驗應脫機，請單擊離線。若要再次使用體驗，必須重新發佈體驗。體驗離線後，將無法再透過其 URL 連線。

注意：離線體驗不會刪除體驗。
