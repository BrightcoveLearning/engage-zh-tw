將自訂網域指派給 Web 體驗

在本主題中，您將學習如何將自訂網域指派給 Web 體驗。

將自訂網域指派給 Web 體驗可讓您透過易於記憶的 URL (與具有site-slug.bcvp0rtal.com URL) 存取網站的方式。使用自訂網域時，您也可以指派 URL 的選用路徑。使用路徑允許您使用不同的模板，製作微網站，宣傳活動，內部門戶網站等，同時仍然使用相同的域。例如，videos.mycustomdomain.com可以擴展為也包含像videos.mycustomdomain.com/event或videos.mycustomdomain.com/product. 路徑允許擁有多個網站和 SSL 的發佈者只需要為一個網域設定 SSL，而不是每個單獨的網站-蛞蝓。

注意：自訂網域必須建立為專案設定的一部分，才能將其指派給體驗。如需詳細資訊，請參閱設定自訂網域和 SSL 。

若要將自訂網域指派給 Web 體驗，請依照下列步驟執行：

  1. 編輯體驗。
  2. 點擊站點配置>網址在左側導航中。將會顯示指派給網站的初始網域。
  3. ( 選擇性) 取消勾選「作用中?」核取方塊以停用site-xxxx.bcvp0rtal.com domain .一般而言，發佈者會在使用自訂網域site-xxxx.bcvp0rtal.com時停用，因此不會由搜尋引擎編製索引。
  4. 按一下 [ 新增自訂網域] 。
  5. 根據預設，網域將處於作用中狀態。取消勾選積極的禁用 Brightcove Engage 在條目上的路由，但它不會釋放路徑/域組合以在另一個站點上使用。
  6. 從下拉式清單中選取網域。將顯示設定為「專案設定」一部分的自訂網域清單。
  7. 選擇性地輸入網域的路徑。
  8. 按一下 [新增自訂網域]，為站台新增其他網域。
  9. 完成後，按一下「儲存」。

按一下刪除圖示 () 以移除自訂網域。

注意：若要讓任何網域變更 (包括啟用或刪除網域) 生效，必須發佈網站。

自訂網域的清單將會列在發佈對話方塊中。

若要停用或移除任何自訂網域，請編輯網站，按一下 [網站設定] > [URL]，然後取消核取 [使用中] 核取方塊以停用網域，或按一下 [刪除] 圖示 () 以移除自訂網域。
