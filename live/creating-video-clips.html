創建視頻剪輯

在本主題中，您將學習如何從現場活動中創建視頻剪輯。

概述

Brightcove 參與™提供從正在進行的和已完成的實時事件的實時事件中創建視頻剪輯的能力。您可能要創建剪輯的原因有幾個：

  • 創建一個簡短的預告片，可以發佈到社交媒體上
  • 要限制您要發布的視頻的長度（大多數平台都限制了視頻的長度）
  • 您發現較短的視頻在社交媒體上的效果更好

筆記

  • 只能在活動結束後的7天內創建剪輯。之後，創建剪輯按鈕將被禁用。
  • 並非所有瀏覽器都支持裁剪。
  • 剪輯將精確到幀前提是編碼器正在為流發送SMPTE時間碼 -否則，它們將在幾幀內保持準確。請注意，為了盡快提供剪輯，首先創建一個段精確的剪輯，並在可用後立即替換為幀精確的剪輯。
  • 無法創建流中開始時間少於10秒的剪輯。
  • 對於通過剪輯實時流創建的視頻，媒體共享將失敗。
  • 無法從AES加密的實時流創建剪輯。

瀏覽剪輯編輯器

要打開剪輯編輯器，請單擊居住在導航標題中，單擊事件名稱，然後單擊創建剪輯在左側導航欄中。剪輯編輯器將打開。

請注意，剪輯界面中的所有時間都是基於Brightcove接收視頻數據的UTC時間。

剪輯編輯器的組織如下：

  • 一種 - 預覽播放器
  • B- 播放控制按鈕
  • C - 剪輯時間
  • D- 剪輯按鈕
  • E- 預覽剪輯按鈕
  • F - 保存按鈕
  • G - 播放頭
  • H - 焦點時間表
  • 一世 - 上下文時間表
  • J- 縮放按鈕
  • K- 剪輯清單

創建一個片段

請按照以下步驟創建一個剪輯。

  1. 使用剪輯按鈕或快捷鍵設置剪輯的起點和終點。
  2. 點擊保存。
  3. 輸入剪輯視頻屬性。
    • 剪輯名稱 -剪輯名稱
    • 簡短的介紹 - 剪輯說明
    • 標籤 - 與剪輯相關聯的標籤
    • 參考編號 -分配給剪輯的參考ID

    任何必填的自定義字段也將顯示。

  4. 點擊創建剪輯。剪輯的處理將開始。

所有剪輯的列表將顯示在剪輯編輯器下方。點擊剪輯名稱會在另一個瀏覽器標籤的“視頻"頁面上打開剪輯。

預覽播放器

預覽播放器用於預覽剪輯。當事件正在進行中並且剪輯編輯器打開時，剪輯將從“安全實時點"開始播放，該“安全實時點"大約在結束前3個片段。請注意，並非所有瀏覽器都支持自動播放。

播放控制按鈕

播放控制按鈕用於控制預覽播放器內部片段的播放。

按鈕 描述

將播放頭後退15秒
從播放頭位置開始播放視頻
將播放頭向前跳躍15秒

剪輯時間

剪輯時間顯示播放頭的當前UTC時間和可剪輯時間。單擊UTC鏈接將切換到本地時間顯示。

剪輯按鈕

剪輯按鈕用於剪輯源視頻的所需部分。

按鈕 描述
跳至視頻提示
設置剪輯的起點。也可以通過拖動焦點時間線開始處的手柄來設置起點。
設置剪輯的終點。也可以通過拖動焦點時間線末端的手柄來設置終點。
跳到視頻的提示點

提示點和提示點是剪輯的起點和終點。這些點中的每一個都用可拖動的手柄在焦點時間軸上表示，兩點之間的時間用綠色填充。這些點在上下文時間軸中也表示為綠色矩形的左邊緣和右邊緣，指示剪輯空間。

預覽片段

單擊預覽剪輯按鈕以播放剪輯。

播放頭

播放頭是玩家播放時間的指示器。播放頭在焦點時間軸中由一條紅線表示，頂部帶有一個紅色圓圈，在上下文時間軸中帶有一條紅色垂直線。

在焦點時間軸上，可以拖動播放頭以在視頻中滑動。雙擊焦點時間軸會將播放頭移到該點。播放頭還可以用於使用提示/提示按鈕或使用鍵盤快捷鍵來設置提示和提示點。

焦點時間表

焦點時間軸是界面中的頂部時間軸。

播放頭是玩家播放時間的指示器。播放頭在焦點時間軸中由一條紅線表示，頂部帶有一個紅色圓圈。

 

上下文時間表

上下文時間線是剪輯編輯器中的底部時間線。上下文時間軸顯示了視頻的整個時長，與縮放級別，剪輯的選定區域，播放頭的位置以及當前的焦點視口無關。時間線用紅線表示。

縮放按鈕

縮放按鈕用於在焦點時間軸上放大或縮小。

按鈕 描述
放大焦點時間線。上下文時間軸中的焦點視口（白框）將顯示焦點時間軸中正在顯示的剪輯區域。
縮小焦點時間線。上下文時間軸中的焦點視口（白框）將顯示焦點時間軸中正在顯示的剪輯區域。

剪輯清單

剪輯列表將顯示在剪輯編輯器下方。點擊剪輯名稱會在另一個瀏覽器標籤的“視頻"頁面上打開剪輯。

鍵盤快捷鍵

鍵盤快捷鍵也可以用於播放視頻並根據視頻創建剪輯。以下鍵盤快捷方式可用。
