概述：創建，編輯和發布Web體驗

在本主題中，您將學習創建，編輯和發布Web Experience的基本步驟。

Brightcove 互動™讓您無需任何技術資源，即可建立引人入勝的視訊體驗。Web體驗遵循響應式設計原則，因此在台式機，平板電腦和移動設備上看起來很棒。

以下是使用Brightcove Engage Web模板構建的Web體驗的示例。

有關構建Web Experience的分步指南，請參閱一步步：創建和發布Web體驗。

創建和發布Web Experience的主要步驟是：

  • 選擇一個模板
  • 添加視頻並自定義體驗
  • 預覽並發布體驗

選擇範本

當您建立新的體驗時，您可以從選擇範本開始。範本控制體驗的整體外觀和感覺。Brightcove Engage提供了一組模板，使您可以靈活地提供與觀眾互動所需的視頻體驗類型。要了解有關模板的更多信息並查看一些示例站點，請參閱體驗模板概述話題。

選擇模板後，您可以選擇預先填充樣本內容的經驗。點擊是的，請預先填寫我的經驗會使用您帳戶中的視頻預先填充體驗。這可讓您在開始自訂程序時更容易看到體驗的外觀。

新增視訊和自訂體驗

創建體驗後，經驗編輯用於自定義體驗。可以完成下列類型的自訂：

  • 頁數 -可以將圖像，文本和鏈接添加到頁面模板中，以根據需要添加品牌體驗。要了解有關向頁面添加對象的更多信息，請參見自定義Web Experience Pages的內容。
  • 影片 -創建收藏集以選擇和組織視頻內容。要了解有關將視頻內容添加到網站的更多信息，請參閱將視頻添加到體驗中。
  • 風格 -可以通過選擇新主題或更改站點顏色和字體來自定義體驗的外觀。要了解有關自定義樣式的更多信息，請參見自定義Web體驗的樣式。
  • 訪問控制 -限制訪問該網站。要了解有關站點配置的更多信息，請參閱控制對體驗的訪問。
  • 概述 -編輯體驗的名稱，描述和語言。要了解有關編輯網站概述的更多信息，請參閱編輯體驗詳細信息。

預覽和發佈體驗

預覽體驗可讓您瞭解體驗在行動裝置、平板電腦和桌上型電腦裝置上的顯示方式。體會 可以在新的瀏覽器選項卡中預覽，也可以生成可共享的鏈接，以便其他人可以預覽站點。發佈體驗會產生網站的靜態 URL，並讓檢視者存取。要了解有關預覽和發布Web Experience的更多信息，請參見預覽和發布Web體驗。
