    逐步：建立和發佈 Web 體驗

    本主題提供一組步驟，您可以遵循以建立和發佈使用 Brightcove Entence™ 的 Web 體驗。

    完成本主題後，您應該能夠：

    • 使用布萊特灣互動打造全新的網路體驗
    • 新增視訊至體驗
    • 自訂體驗
    • 預覽和發佈體驗
    注意：有關創建移動應用體驗的分步指南，請參閱一步步：建立和發佈行動應用程式體驗。

    觀眾

    任何人誰需要創建網絡視頻門戶網站顯示視頻內容。

    先決條件

    • Brightcove參與帳戶（註冊試用）

    開始之前

    本逐步指南將使用 Brightcove Ent習試用帳戶隨附的範例影片。隨意上傳和使用您自己的視頻。

    建立網路體驗

    若要使用布萊特灣參與建立網路體驗，您將完成下列工作：

    • 使用播放清單整理影片
    • 創造新的Web體驗
    • 編輯網站詳細資料
    • 將圖像和文本添加到頁面模板
    • 建立收藏集並將視訊新增至收藏
    • 預覽體驗
    • 變更工址樣式
    • 發佈體驗

    使用播放清單組織影片

    在您開始建立體驗之前，請先考慮您想要如何組織或群組您的影片。一般而言，您會想要將相關影片分組在一起。例如，您可能希望組 CEO 視頻在一起，人力資源視頻在一起，營銷視頻等。當觀眾使用您的網路體驗時，他們會看到這些群組的清單、選取群組，然後選擇想觀看的影片。使用「布萊特灣互動」時，相關影片會使用「播放清單」分組。

    注意：如果您使用自己的影片，請建立數個播放清單來整理影片。

    若要將範例視訊整理到播放清單中，請依照下列步驟執行。

    1. 登入布萊特灣參與。將會出現 [ 專案 ] 頁面。
    2. 按一下導覽標題中的 [ 影片 ] 索引標籤。將會出現範例視訊清單。
    3. 在左側導覽中，按一下 [ + 建立播放清單 ]。
    4. 將播放清單類型設定為「手動」。
    5. 將播放清單名稱設定為Featured Video。
    6. 勾選「在側邊欄中顯示播放清單」。
    7. 按一下「儲存」。
    8. 使用相同的設定，再建立兩個播放清單，以Sales Videos及HR Videos。您應該建立三個播放清單。
    9. 在左側導覽中，按一下 [ 所有視訊 ]。
    10. 按一下 2019 全手會議影片旁的核取方塊，然後按一下 [ 更多 > 新增至播放清單 ]。
    11. 選取精選影片播放清單，然後按一下「新增」。
    12. 選擇下列影片，並將其新增至「銷售影片」播放清單。
      • 2018 年銷售策略會議
      • 銷售指南-在新市場開發您的管道-第 02 集
      • 銷售指南-追加銷售遊覽的新技術-Ep 01
    13. 選取下列影片，然後將它們新增至 HR 影片播放清單。
      • 人力資源-建立您的內部通訊計畫-E02
      • 人力資源-新員工上任-E01

    所有影片現在都會整理成播放清單。按一下左側導覽中的播放清單名稱會顯示該播放清單中的影片。

    如需有關播放清單的詳細資訊，請參閱建立和管理播放清單主題。

    建立全新的 Web 體驗

    現在影片已經整理成播放清單，我們就可以建立新的體驗。若要建立新的 Web 體驗，請依照下列步驟執行。

    1. 在導覽標題中，按一下 [ 專案] 索引標籤。
    2. 確認會顯示「建立體驗」頁面。如果出現體驗清單，請按一下 [ 新體驗 ] 以顯示 [ 建立體驗 ] 頁面。
    3. 找到網頁範本然後點擊選擇。
      注意：一旦您選取範本，就無法變更。有關模板的更多信息，請參見體驗模板概述話題。
    4. 輸入示例Web模板體驗為了經驗標題。為了預先填充體驗？設定中，選取 [ 不，謝謝 ]。
    5. 按一下「建立體驗」。

    體驗編輯器將會開啟。體驗編輯器可用來自訂體驗，並提供自訂頁面配置、新增影片、選擇樣式和設定其他設定的功能。所有自訂設定均可使用左側導覽功能表存取。

    編輯網站詳細資料

    體驗標題、說明和語言設定可以使用體驗編輯器編輯。

    1. 按一下左側導覽中的概觀。
    2. 為了經驗描述，添加示例Web體驗。
    3. 按一下「儲存」。

    將影像和文字新增至頁面範本

    每個 Web 模板提供了一組可以自定義的頁面模板。Web模板提供家，類別和影片詳細資料頁面模板。這些頁面可以在左邊界看到作為頁面選單的一部分。

    頁面模板將在頁面上顯示佔位符和鉛筆圖標（）表示可以編輯該對象。這些佔位符可用於添加圖像，文本和鏈接到您的頁面。

    有關將對象添加到頁面模板的更多信息，請參見將圖像，文本和鏈接添加到Web體驗頁面。

    若要將物件新增至頁面範本，請依照下列步驟執行。

    1. 按一下左側導覽中的「 > 頁面首頁」。鉛筆圖標（）指示頁面上可以添加圖像/文本/鏈接的區域。您可以按一下頁面上的靜態文字物件來編輯它們。
    2. 在頁面頂部，單擊+添加內容。
    3. 點擊上載選項，然後單擊保存。選擇中上載將圖像佔位符添加到頁面。
    4. 點擊+添加照片。
    5. 請點擊選擇文件然後瀏覽並選擇要用於標題的圖像。請點擊保存。
      注意：布萊特灣將調整圖像大小為 140 像素 x 50 像素。
    6. 確認影像顯示在頁面標頭中。
    7. 在頁面底部，單擊持票人在頁腳的左側。輸入您的公司名稱，然後單擊保存。
    8. 點擊類別和影片詳細資料左邊距中的頁面。請注意，頁首影像和標題文字也會出現在這些頁面上。返回「首頁」。

    建立收藏集並將視訊新增至收藏

    使用體驗編輯器，您可以選擇要在體驗中顯示哪些視訊。建立體驗時，視訊會組織成集合。在此示例中，收藏集將基於之前的播放列表較早建立。

    如需有關建立集合和將視訊新增至體驗的詳細資訊，請參閱將視訊新增至體驗。

    若要將視訊新增至體驗，請依照下列步驟執行。

    1. 按一下左側導覽中的 [ 影片 ]。
    2. 命名第一個收藏營業額然後按輸入。
    3. 按一下集合名稱以開啟集合編輯器。
    4. 選取「播放清單」以查看條件。
    5. 點擊銷售影片播放列表。該播放清單中的影片會出現在影片清單中。按一下「儲存」。
      注意：使用播放清單視訊指派時，視訊的順序會根據「視訊」頁面中設定的播放清單設定而定。
    6. 使用 HR 影片播放清單作為準則，建立另一個名為 HR 的集合。
    7. 確認已建立 2 個收藏。
      注意：您可以透過按一下集合並將集合拖曳至新位置來變更集合的順序。

    預覽體驗

    預覽體驗可讓您查看網站在行動裝置、平板電腦和桌上型電腦裝置上的顯示方式。您會想要預覽體驗，當您完成自訂的步驟。您可以在新的瀏覽器索引標籤中預覽體驗，也可以產生可共用的連結，讓其他人可以預覽入口網站。有關預覽和發布Web體驗的更多信息，請參見預覽和發布Web體驗。

    若要預覽體驗，請依照下列步驟執行。

    1. 編輯體驗時，單擊預習然後點擊響應式預覽。
    2. 預覽會在新的瀏覽器索引標籤中開啟。
    3. 關閉網站預覽標籤。

    變更工址型式

    「樣式」設定可用來自訂使用體驗的網站顏色和字型。有關更改網站樣式的更多信息，請參見自定義Web體驗的樣式。

    若要變更網站樣式，請依照下列步驟執行。

    1. 按一下左側導覽中的「樣式」。
    2. 「主題」是一組可套用至範本的顏色和字型。選取「主題」。本示例將使用藍色陰影主題。
    3. 選取要使用的「主要」和「次要」字體。的網站標題字體控制標題中使用的標題字體。如果您遵循這個逐步文件，則會先前將影像新增至標頭。
    4. 點擊<返回體驗樣式菜單頂部的鏈接。

    發佈體驗

    發佈 Web 體驗可在公用 URL 中使用。有關預覽和發布門戶體驗的更多信息，請參見預覽和發布Web體驗。

    若要發佈體驗，請依照下列步驟執行。

    1. 在“體驗編輯器”內部，單擊發布。
    2. 在“發布站點”對話框上，單擊立即發布確認出版物。

    點擊網站 URL，在其他瀏覽器標籤中打開它。
