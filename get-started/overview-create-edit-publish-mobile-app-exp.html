概述：創建，編輯和發布移動應用程序體驗

在本主題中，您將學習創建，編輯和發布Mobile App Experience的基本步驟。

Brightcove 互動™讓您無需任何技術資源，即可建立引人入勝的視訊體驗。以下是使用Brightcove Engage移動應用程序模板構建的移動應用程序體驗的示例。

有關構建移動應用體驗的分步指南，請查看一步步：創建和發布移動應用體驗。

創建和發布移動應用體驗的主要步驟是：

  • 選擇一個模板
  • 添加視頻並自定義體驗
  • 預覽並發布體驗

選擇範本

當您建立新的體驗時，您可以從選擇範本開始。範本控制體驗的整體外觀和感覺。要了解有關模板的更多信息並查看一些示例站點，請參閱體驗模板概述話題。

選擇模板後，您可以選擇預先填充樣本內容的經驗。點擊是的，請預先填寫我的經驗會使用您的Brightcove帳戶中的視頻預先填充體驗。這可讓您在開始自訂程序時更容易看到體驗的外觀。

新增視訊和自訂體驗

創建體驗後，經驗編輯用於自定義體驗的不同方面。可以完成下列類型的自訂：

  • 頁數 -可以將徽標添加到主頁，以根據需要對體驗進行品牌化。要了解有關添加徽標的更多信息，請參見向移動應用體驗添加徽標。
  • 影片 -創建收藏集以選擇和組織視頻內容。要了解有關將視頻內容添加到體驗中的更多信息，請參閱將視頻添加到體驗中。
  • 風格 -可以通過選擇新主題或更改站點顏色來自定義體驗的外觀。要了解有關自定義樣式的更多信息，請參見造型移動應用程序的體驗。
  • 訪問控制 -您可以使用“單一登錄"來限制對已發布應用的訪問。要了解有關訪問控制的更多信息，請參閱控制對體驗的訪問。
  • 概述 -您可以編輯體驗的名稱，描述和語言。要了解有關編輯網站概述的更多信息，請參閱編輯體驗詳細信息。

預覽和發佈體驗

預覽移動應用程序體驗可讓您查看體驗如何在移動設備上顯示。體會 可以在新的瀏覽器選項卡中預覽，也可以生成可共享的鏈接，以便其他人可以預覽站點。發布體驗會生成訪問密鑰，而訪問密鑰是Brightcove Engage移動應用程序。要了解有關預覽和發布體驗的更多信息，請參閱預覽和發布移動應用程序體驗。
